# 🧠 Computer Quiz Game (Python)

A simple interactive **quiz game** built in Python.  
Test your knowledge of computer-related terms and general abbreviations. The game asks multiple questions, checks your answers, and keeps track of your score.

---

## ✨ Features
- Interactive quiz with multiple questions.
- Input validation for user responses.
- Score tracking throughout the game.
- Ends with your final score.
